**ЭТАП 2: КРИТИЧЕСКИЙ АНАЛИЗ (только для FAILED_SALE + MANAGER_FAULT)**

Выполни детальный анализ ошибки менеджера:

**1. Определи конкретный код ошибки** (используй коды ошибок M1-M9)

**2. Извлеки ключевые факты:**
- Что хотел клиент?
- Какая была цена?
- Как можно было спасти сделку?

**3. Сформулируй конкретное действие для спасения сделки**

**ОБЯЗАТЕЛЬНЫЙ ФОРМАТ ОТВЕТА (JSON):**

```json
{
  "status": "alert",
  "error_code": "M1",
  "error_description": "Краткое описание ошибки",
  "client_wanted": "Что хотел клиент",
  "price_mentioned": "Названная цена",
  "context": "Ключевой контекст ситуации",
  "solution": "Как можно было спасти сделку"
}
```

**ВНИМАНИЕ:** Если это НЕ FAILED_SALE с MANAGER_FAULT, верни {"status": "ignore"}
